# Contributing to TaskMaster

Thank you for considering contributing to TaskMaster! We welcome contributions from everyone.

## How to Contribute

1. **Fork the Repository**: Click the "Fork" button on the top right of the repository page.

2. **Create a New Branch**: Create a feature or bugfix branch to work on.
   ```bash
   git checkout -b feature/your-feature-name
   ```

3. **Make Changes**: Implement your changes.

4. **Commit Your Changes**: Commit with a clear and concise message.
   ```bash
   git commit -m "Add feature XYZ"
   ```

5. **Push to Your Fork**: Push your changes to your forked repository.
   ```bash
   git push origin feature/your-feature-name
   ```

6. **Open a Pull Request**: Go to the original repository and open a pull request.

## Code Style

- Follow PEP 8 for Python code.
- Use Prettier for formatting JavaScript/TypeScript code.

## Testing Requirements

Ensure all tests pass before submitting a pull request. Add new tests to cover your changes.
